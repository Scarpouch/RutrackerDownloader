**Rutracker Downloader** - Утилита для скачивания видео с YouTube, Rutube, Twitch и других платформ через yt-dlp.

Для скачивания всех файлов, нажмите зелёную кнопку «<> Code» на главной странице репозитория и выберите «Download ZIP»

**Быстрый старт:**
1. Распакуйте архив
2. Запустите `START.bat`
3. В интерфейсе вставьте ссылку на видео с любого ресурса
4. Выберите папку для сохранения
5. Нажмите "Скачать"
6. Получаем результат

**Функционал:**
- Скачивание видео до 1080p (best[height<=1080])
- Проверка доступности URL перед скачиванием
- Проверка существования файла (с запросом на создание дубля)
- Прогресс-бар и статус в реальном времени
- Автосохранение настроек окна и папки
- Логирование в `data/logs/`
- Скрытые окна yt-dlp
