**Rutracker Downloader** - Утилита для скачивания видео с YouTube, Rutube, Twitch и других платформ через yt-dlp.

**Быстрый старт:**
1. Скачайте `yt-dlp.exe` с https://github.com/yt-dlp/yt-dlp
2. Поместите в папку с `downloader.py` или добавьте в PATH
3. Запустите `START.bat`
4. Вставьте ссылку на видео
5. Выберите папку сохранения
6. Нажмите "Скачать"

**Функционал:**
- Скачивание видео до 1080p (best[height<=1080])
- Проверка доступности URL перед скачиванием
- Проверка существования файла (с запросом на дубль)
- Прогресс-бар и статус в реальном времени
- Автосохранение настроек окна и папки
- Логирование в `data/logs/`
- Поддержка Windows (скрытое окно yt-dlp)
